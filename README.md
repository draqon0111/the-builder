# 🏗️ Responsive Builder Website

A simple responsive builder-style website created using **HTML and CSS**.  
This project demonstrates a basic layout with a header, navigation bar, main content area, sidebar, and footer.

---

## 📌 Features

- ✅ Responsive design using CSS media queries  
- ✅ Simple and clean layout  
- ✅ Navigation bar with hover effects  
- ✅ Flexbox-based structure  
- ✅ Mobile-friendly view  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Flexbox & Media Queries)
